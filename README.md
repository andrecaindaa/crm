# CRM – Estágio 

CRM desenvolvido em Laravel 12 + Vue para gestão de entidades, contactos,
negócios e atividades comerciais.

## Stack
- Laravel 12
- Vue 3 + Vite
- MySQL
- ShadcnVue

## Funcionalidades principais
- Gestão de Entidades e Pessoas
- Pipeline de Negócios (Kanban)
- Calendário e Atividades
- Timeline unificada por negócio
- Envio de propostas por email
- Follow-ups automáticos
